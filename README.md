# Figma Testcase Generator

## Kurulum

1. Projeyi klonlayın:
   ```bash
   git clone https://github.com/nurullahsahin44/figma-testcase-generator.git
   ```

2. Gerekli bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

## Kullanım

1. Uygulamayı başlatın:
   ```bash
   npm start
   ```

2. Figma dosyasınızı yükleyin ve otomatik olarak testcase'leri oluşturun.
